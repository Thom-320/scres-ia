## New prospective research target

preregistered test of incremental hybrid value:

`LCB95[V(hybrid) - V(best structured control)] >= 0.01`

The primary scale should be frozen before development (recommended: absolute
ReT difference on the [0, 1] scale). The structured comparator must receive the
same operational information, intervention rights, online compute budget, and
physical resources. Environment or risk parameters may not be selected using
hybrid returns.

The candidate mechanism to assess is a learned long-horizon continuation value
and/or learned belief correction embedded in short-horizon constrained
belief-MPC. MPC remains responsible for immediate feasibility and product
service constraints. This is a prospective method hypothesis, not an executed
result.
