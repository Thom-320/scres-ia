## Binding identity

**Primary journal:** *Computers & Industrial Engineering*.

**Working title:** *Learning-Augmented Event-Triggered MPC for Supply-Chain

The manuscript will wait for one prospective, fail-closed attempt at the

## Current truth versus prospective claim

Program Q is terminal. It independently confirmed that frozen recurrent
policies beat every one of 65,536 open-loop calendars and are practically
equivalent in canonical ReT to the strongest tested structured controller.
It did not establish a neural premium or worst-product-fill non-inferiority.
Those facts are the empirical foundation and motivation, not evidence that the
new hybrid already works.

The proposed hybrid remains prospective. Its central claim is authorized only
if it beats the exact static frontier, pure recurrent RL, and all preregistered
model-based controllers under matched information, physical resources, review
rights, and online computation while preserving worst-product service.

## Scientific contribution

The method closes the DES decision loop through four separable components:

1. a causal recurrent belief over latent demand/disruption conditions;
2. a distributional terminal ReT continuation value;
3. constrained scenario MPC selecting product mix and the next review time;
4. a calibrated confidence gate that falls back to feasible robust MPC.

The contribution is not the coexistence of RL and MPC. It is the controlled
attribution of belief, continuation value and endogenous intervention timing,
with exact static benchmarking, strong model-based comparators, resource and
information matching, and direct full-DES verification.

## Binding order

1. Run U0 static discovery as a non-blocking benchmark without exposing the
   exhaustive answer matrix to any search method.
2. Strengthen the ReT-aligned full-DES MPC on burned artifacts.
3. If the strengthened comparator absorbs the current opportunity, run the
   corrected S1b nonstationary risk cascade and select the least-severe
   connected region using only pre-learner headroom.
4. Prove endogenous-review value relative to the strongest **fixed-cadence
   adaptive controller with the same four-review budget** before training an
   event-triggered learner. The unconstrained weekly perfect-information
   oracle is a ceiling and is reported as regret; a four-review policy is a
   subset of that weekly action class and therefore cannot dominate it.
5. Run nested hybrid ablations only after every pre-learner gate passes.
6. Freeze architecture, compute budget, margins, comparators and seeds before
   opening virgin confirmation.
