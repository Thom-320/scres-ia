# Legacy Materials

This directory stores historical or ad-hoc development artifacts that are not part of the current paper-facing benchmark contract.

## Subdirectories

- `patches/`: one-off migration/edit scripts used during rapid iteration.
- `debug_tests/`: root-level exploratory checks kept for historical reference.

The active repository surface should be read from:

- `supply_chain/`
- `scripts/`
- `tests/`
- `docs/`
